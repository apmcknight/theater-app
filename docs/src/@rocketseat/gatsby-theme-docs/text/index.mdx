# Introduction

Eventily is a Customer Mangment System built for Event Services, Theater's, and Live Production venues.

## Features at a glance:

- 🖥 Built using the ASP.NET framework in C#, with Enitiy Core and SQLite (by default)¡
- 📊 Can use both SQLite and SQL as data-storage
- 📝 Fully documented software
- 📱 Responsive and mobile friendly
- ⚡️ Works offline (with limited features)
- and much more 🔥
